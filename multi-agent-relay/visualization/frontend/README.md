# 多智能体可视化平台 - 前端

## 简介

前端是一个单页面应用，通过 WebSocket 与后端实时通信，展示多智能体的协作过程。

## 功能特性

- 实时显示4个智能体的状态
- 可视化展示接力流程
- 任务提交和管理
- 执行日志实时滚动

## 技术栈

- HTML5
- CSS3 (原生)
- JavaScript (原生，无需框架)
- WebSocket API

## 文件结构

```
frontend/
├── index.html      # 主页面
├── style.css       # 样式（已内联到后端HTML中）
├── app.js          # 逻辑（已内联到后端HTML中）
└── README.md       # 本文件
```

## 访问方式

启动后端后，打开浏览器访问:
```
http://localhost:8080/dashboard
```
