# 多智能体接力系统 + 可视化平台

一个完整的**局域网多智能体协作框架**，支持4个角色按流程接力完成任务，并提供实时可视化界面观察整个协作过程。

## 🎯 项目结构

```
multi-agent-relay/
├── agents/                      # 智能体角色定义
│   ├── analyzer.md             # 📊 分析员 - 理解需求
│   ├── planner.md              # 📋 策划师 - 制定计划
│   ├── executor.md             # 🔨 执行者 - 执行产出
│   └── reviewer.md             # ✅ 审核员 - 质量检查
│
├── relay_system.py             # 核心接力逻辑
├── demo_task.md                # 演示任务定义
├── start.sh                    # 🚀 一键启动脚本
├── README.md                   # 本文件
│
└── visualization/              # 可视化平台
    ├── backend/
    │   ├── main.py             # FastAPI + WebSocket 后端
    │   └── README.md
    └── frontend/
        └── README.md           # 前端说明
```

## 🚀 快速开始

### 1. 一键启动可视化平台

```bash
cd multi-agent-relay
./start.sh
```

然后打开浏览器访问: **http://localhost:8080/dashboard**

### 2. 运行命令行演示

```bash
python3 relay_system.py
```

## 🔄 接力流程

| 阶段 | 角色 | 输入 | 输出 |
|------|------|------|------|
| 1 | 📊 分析员 | 用户任务 | 需求分析报告 |
| 2 | 📋 策划师 | 分析报告 | 详细执行计划 |
| 3 | 🔨 执行者 | 执行计划 | 任务成果 |
| 4 | ✅ 审核员 | 执行结果 | 质量评估报告 |

## 🔧 技术栈

- **后端**: Python + FastAPI + WebSocket
- **前端**: HTML5 + CSS3 + JavaScript (原生)
- **通信**: WebSocket 实时推送
- **部署**: 纯 Python，无需额外服务
